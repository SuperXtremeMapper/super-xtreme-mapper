Installation Instructions

Your download should have started automatically. Since SXM is not signed with an Apple Developer certificate, macOS Gatekeeper will initially block the app. Follow these steps to install:

  1. 1
    Open the .dmg file

    Double-click the downloaded file to mount the disk image.

  2. 2
    Drag to Applications

    Drag the Super Xtreme Mapper app to your Applications folder.

  3. 3
    First launch (important!)

    Right-click (or Control-click) the app in Applications and select Open from the context menu.

  4. 4
    Confirm opening

    A dialog will appear saying the app is from an unidentified developer. Click Open to proceed.

  5. 5
    You're done!

    SXM will remember this and open normally in the future.
