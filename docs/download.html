Installation

Since SXM isn't signed with an Apple Developer certificate, macOS will initially block it. Here's how to install:

1
Open the DMG

Double-click the downloaded file to mount it

2
Drag to Applications

Copy the app to your Applications folder

3
Right-click to Open

Control-click the app and select Open

4
Confirm

Click Open in the security dialog

Alternative: Go to System Settings → Privacy & Security and click "Open Anyway"
